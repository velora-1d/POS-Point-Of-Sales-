# Flow: Laporan Karyawan

**Role**: Owner, Supervisor

## Laporan Kehadiran

```
Owner/Supervisor buka Laporan → Karyawan → Kehadiran
  → Pilih karyawan & bulan
  → Lihat: hadir, terlambat, absen, cuti per hari
  → Persentase kehadiran
  → Export PDF/Excel untuk payroll
```

## Laporan Performa Kasir

```
Owner buka Laporan → Karyawan → Performa
  → Lihat per kasir per shift:
      → Jumlah transaksi
      → Total revenue
      → Rata-rata nilai transaksi
      → Jumlah refund/void
      → Selisih kas (jika ada)
  → Ranking kasir by performance
```
