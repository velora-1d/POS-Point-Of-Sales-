# Flow: Manajemen Meja & QR Code

**Role**: Owner (setup), Semua (lihat status)

## Setup Meja

```
Owner buka menu Meja
  → Tambah meja: nama, kapasitas, posisi di layout
  → Drag & drop atur posisi layout visual
  → Generate QR code per meja (otomatis)
  → Print QR code → tempel di meja fisik
```

## Status Meja Real-time

```
Semua role lihat layout meja
  → Hijau = Available
  → Merah = Occupied (ada order aktif)
  → Kuning = Reserved (ada reservasi)
  → Klik meja → lihat order aktif di meja tersebut
```

## Customer Scan QR Meja

```
Customer scan QR meja
  → Buka halaman menu outlet
  → Bisa: lihat menu, pesan, lihat status order, edit pesanan
  → Tidak perlu login / daftar
```
