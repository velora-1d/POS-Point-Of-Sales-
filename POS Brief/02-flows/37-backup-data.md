# Flow: Backup & Keamanan Data

**Role**: Owner only

## Lihat Status Backup

```
Owner buka Pengaturan → Backup & Keamanan
  → Lihat backup terakhir: tanggal, ukuran, status
  → Lihat jadwal backup otomatis
  → Download backup manual (opsional)
```

## Manual Backup

```
Owner klik "Backup Sekarang"
  → Sistem generate backup database
  → Upload ke cloud storage (encrypted)
  → Notif selesai ke owner
```

## Restore (jika diperlukan)

```
Hanya bisa dilakukan oleh tim devops
  → Owner hubungi support
  → Pilih titik restore (tanggal backup)
  → Konfirmasi → data di-restore
```
