# Flow: Manajemen Outlet & Cabang

**Role**: Owner

## Tambah Outlet

```
Owner buka menu Outlet
  → Klik Tambah Outlet
  → Input: nama, alamat, nomor telepon
  → Setting per outlet:
      → Workflow status order (kustomisasi tahapan)
      → Default struk (print/WA/skip)
      → Bar approval aktif/tidak
      → Customer bisa lihat status/tidak
  → Simpan → outlet aktif
```

## Setting Per Outlet

```
Owner pilih outlet
  → Edit setting spesifik outlet
  → Assign karyawan ke outlet
  → Set harga per outlet (jika beda)
  → Lihat laporan per outlet
```

## Perbandingan Outlet

```
Owner buka Dashboard Multi Outlet
  → Lihat semua outlet dalam 1 tampilan
  → Bandingkan: revenue, order, karyawan aktif
```
