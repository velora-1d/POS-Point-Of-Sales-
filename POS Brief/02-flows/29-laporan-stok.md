# Flow: Laporan Stok & Inventori

**Role**: Owner, Supervisor

## Laporan Mutasi Stok

```
Owner buka Laporan → Stok → Mutasi
  → Pilih periode & produk/bahan baku
  → Lihat: stok awal, masuk, keluar, penyesuaian, stok akhir
  → Detail per transaksi yang pengaruhi stok
```

## Laporan Stok Opname

```
Owner buka Laporan → Stok → Opname
  → Pilih tanggal opname
  → Lihat selisih per produk: sistem vs aktual
  → Nilai kerugian/kelebihan stok
```

## Laporan Expired

```
Owner buka Laporan → Stok → Expired
  → Lihat semua produk/bahan baku yang sudah expired
  → Total kerugian dari stok expired
  → Filter per periode
```

## Laporan HPP

```
Owner buka Laporan → Produk → HPP
  → Lihat HPP vs harga jual per produk
  → Margin per produk
  → Produk dengan margin terendah/tertinggi
```
