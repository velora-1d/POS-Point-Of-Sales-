# Flow: Export & Import Produk

**Role**: Owner

## Export Katalog Produk

```
Owner buka Produk → Export
  → Pilih format: Excel / CSV
  → Download file katalog produk lengkap:
      → Nama, kategori, harga, HPP, stok, status
```

## Import Produk (Bulk)

```
Owner buka Produk → Import
  → Download template Excel
  → Isi data produk di template
  → Upload file
  → Sistem validasi:
      → Format kolom benar?
      → Nama produk duplikat?
      → Harga valid?
  → Preview hasil import
  → Konfirmasi → produk ditambahkan / diupdate
  → Laporan hasil import (berhasil / gagal per baris)
```
