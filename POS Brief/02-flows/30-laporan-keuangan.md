# Flow: Laporan Keuangan (Mini ERP)

**Role**: Owner only

## Ringkasan Keuangan

```
Owner buka Laporan → Keuangan → Ringkasan
  → Revenue total
  → HPP total
  → Gross profit
  → Pengeluaran operasional
  → Net profit estimasi
  → Piutang outstanding (kasbon + PO belum lunas)
```

## Laporan Piutang

```
Owner buka Laporan → Keuangan → Piutang
  → List semua kasbon + PO outstanding
  → Total piutang per customer
  → Aging piutang (< 30 hari, 30-60 hari, > 60 hari)
  → Export untuk follow-up
```

## Pengeluaran Operasional

```
Owner buka Laporan → Keuangan → Pengeluaran
  → Input pengeluaran: kategori, nominal, tanggal, keterangan
  → Lihat total pengeluaran per kategori
  → Perbandingan pengeluaran vs periode sebelumnya
```

## Rekap Kas Per Shift

```
Owner buka Laporan → Keuangan → Kas Shift
  → List semua shift + rekap kas
  → Filter: kasir, periode, outlet
  → Highlight shift yang ada selisih kas
```
