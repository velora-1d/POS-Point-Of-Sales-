# Flow: Laporan Multi Outlet

**Role**: Owner only

## Dashboard Multi Outlet

```
Owner buka Dashboard → pilih "Semua Outlet"
  → Lihat semua outlet dalam 1 tampilan:
      → Revenue per outlet (hari ini / minggu / bulan)
      → Jumlah order per outlet
      → Produk terlaris per outlet
      → Karyawan aktif per outlet
  → Klik outlet tertentu → zoom ke detail outlet itu
```

## Perbandingan Outlet

```
Owner buka Laporan → Multi Outlet → Perbandingan
  → Pilih periode
  → Tabel perbandingan semua outlet:
      → Revenue, order, avg order value
      → Growth vs periode sebelumnya
  → Chart bar perbandingan visual
  → Export data perbandingan
```
