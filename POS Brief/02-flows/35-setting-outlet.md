# Flow: Setting Outlet

**Role**: Owner only

## Konfigurasi Workflow Order

```
Owner buka Pengaturan → Outlet → Workflow
  → Edit tahapan status order
      → Tambah/hapus/rename tahapan
      → Drag & drop urutan
  → Toggle: bar approval aktif/tidak
  → Toggle: customer bisa lihat status
  → Toggle: customer bisa edit pesanan
  → Simpan → berlaku untuk outlet ini
```

## Konfigurasi Printer

```
Owner/Supervisor buka Pengaturan → Printer
  → Pilih jenis printer: thermal / dot matrix
  → Set koneksi: USB / jaringan (IP)
  → Test print
  → Set default struk: print / WA / skip
```

## Konfigurasi Payment Gateway

```
Owner buka Pengaturan → Payment Gateway
  → Pilih provider
  → Input API key & secret
  → Setup webhook URL
  → Test koneksi
  → Aktifkan metode bayar
```
