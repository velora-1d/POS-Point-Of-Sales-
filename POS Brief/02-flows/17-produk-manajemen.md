# Flow: Manajemen Produk & Kategori

**Role**: Owner, Supervisor (terbatas)

## Tambah Produk

```
Owner buka menu Produk → Tambah Produk
  → Isi detail: nama, deskripsi, foto, kategori
  → Input harga dasar & HPP
  → Setting stok: track stok? track expired?
  → Tambah varian (opsional)
  → Tambah multi harga (opsional)
  → Tambah resep/bahan baku (opsional)
  → Simpan → produk aktif di menu
```

## Toggle Ketersediaan

```
Owner/Supervisor buka produk
  → Toggle "Tersedia" ON/OFF
  → OFF → produk tidak tampil di menu kasir & QR meja
  → ON → tampil kembali
```

## Manajemen Kategori

```
Owner buka menu Kategori
  → Tambah / edit / nonaktifkan kategori
  → Drag & drop untuk ubah urutan tampil di menu
```
