# Flow: Manajemen Karyawan

**Role**: Owner, Supervisor (terbatas)

## Tambah Karyawan

```
Owner buka menu Karyawan
  → Tambah Karyawan baru
  → Input: nama, email, HP, role, outlet, tanggal bergabung
  → Set password awal
  → Set approval PIN (untuk transaksi yang butuh approval)
  → Simpan → karyawan bisa login
```

## Jadwal Shift

```
Supervisor buka Jadwal
  → Pilih karyawan + tanggal
  → Assign template shift
  → Bisa bulk assign (1 minggu sekaligus)
```

## Nonaktifkan Karyawan

```
Owner nonaktifkan karyawan yang resign
  → Karyawan tidak bisa login
  → Data historis tetap tersimpan
```
