# Flow: Download & Print QR Meja

**Role**: Owner, Supervisor

## Download QR per Meja

```
Owner/Supervisor buka menu Meja
  → Pilih meja yang diinginkan
  → Klik tombol "QR Code"
  → Modal QR tampil:
      → Preview QR code
      → Nama meja + nomor meja
  → Pilih format download:
      → PNG → file gambar resolusi tinggi (300 DPI)
      → PDF → layout siap cetak (ukuran A6 / kartu / custom)
  → Klik Download → file tersimpan ke device
```

## Download QR Semua Meja Sekaligus

```
Owner buka menu Meja
  → Klik "Download Semua QR"
  → Pilih format: PNG per file (ZIP) / PDF 1 file (semua meja)
  → Generate → download otomatis
```

## Print QR Langsung

```
Owner/Supervisor klik "Print QR"
  → Pilih printer yang terhubung
  → Pilih ukuran: A4 / A6 / 10x10cm
  → Preview layout sebelum print
  → Klik Print → kirim ke printer
```

## Layout QR yang Digenerate

```
Setiap QR berisi:
  → QR code (besar, mudah di-scan)
  → Nama outlet
  → Nama / nomor meja
  → Teks: "Scan untuk pesan & bayar"
  → Logo outlet (opsional)
```

## Regenerate QR

```
Owner/Supervisor klik "Regenerate QR"
  → Konfirmasi: "QR lama akan invalid"
  → Generate token baru
  → QR lama langsung tidak bisa dipakai
  → Download QR baru
```
