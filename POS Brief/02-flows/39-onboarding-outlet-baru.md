# Flow: Onboarding Outlet Baru

**Role**: Owner

## Setup Awal Outlet Baru

```
1. Owner buat outlet baru
   → Input nama, alamat, telepon
   → Simpan

2. Setup produk & kategori
   → Copy dari outlet yang sudah ada (opsional)
   → Atau input ulang manual

3. Setup meja & QR
   → Tambah meja sesuai layout fisik
   → Generate & cetak QR per meja

4. Setup karyawan
   → Tambah karyawan baru atau assign dari outlet lain
   → Set role & PIN approval

5. Setup harga per outlet (jika beda)
   → Override harga tertentu untuk outlet ini

6. Konfigurasi outlet
   → Workflow status order
   → Setting printer
   → Payment gateway
   → Notifikasi

7. Test transaksi
   → Buat order test
   → Proses pembayaran test
   → Cek laporan muncul

8. Go live!
```
