# Flow: Laporan Member & Loyalty

**Role**: Owner

## Pertumbuhan Member

```
Owner buka Laporan → Member → Pertumbuhan
  → Grafik member baru per bulan
  → Total member aktif
  → Member per tier (Bronze/Silver/Gold)
  → Registrasi via channel (kasir/QR/app)
```

## Analitik Loyalty

```
Owner buka Laporan → Member → Loyalty
  → Total poin dikeluarkan vs diredeem
  → Produk paling sering diredeem
  → Customer paling aktif (poin tertinggi)
  → Poin yang akan expire (jika ada expiry)
```

## Customer Analytics

```
Owner buka Laporan → Customer
  → New vs returning customer
  → Average spending per customer
  → Customer dengan kasbon terbesar
  → Customer paling sering transaksi
```
