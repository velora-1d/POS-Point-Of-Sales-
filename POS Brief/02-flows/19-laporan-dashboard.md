# Flow: Laporan & Dashboard

**Role**: Owner (full), Supervisor (terbatas)

## Dashboard Harian

```
Owner buka Dashboard
  → Lihat ringkasan hari ini:
      → Total revenue
      → Jumlah order
      → Rata-rata nilai order
      → Produk terlaris
      → Shift aktif
  → Alert: stok menipis, expired, approval pending, kasbon jatuh tempo
```

## Laporan Penjualan

```
Owner buka Laporan Penjualan
  → Pilih periode (harian/mingguan/bulanan/custom)
  → Filter: outlet, kasir, kategori, produk
  → Lihat: revenue, order, diskon, net revenue
  → Breakdown per metode bayar
  → Chart tren penjualan
```

## Export Laporan

```
Owner klik Export
  → Pilih jenis laporan
  → Pilih format: PDF / Excel
  → Pilih periode
  → Generate → download link tersedia 24 jam
```

## Laporan Multi Outlet

```
Owner buka Perbandingan Outlet
  → Lihat performa semua outlet dalam 1 tampilan
  → Bandingkan revenue, order, produk terlaris per outlet
```
