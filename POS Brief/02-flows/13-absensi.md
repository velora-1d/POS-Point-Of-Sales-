# Flow: Absensi Karyawan

**Role**: Semua role (clock in/out sendiri), Supervisor/Owner (kelola)

## Clock In

```
Karyawan datang
  → Buka aplikasi → klik "Clock In"
  → Sistem record: waktu, user, outlet
  → Jika terlambat dari jadwal → status: LATE
  → Jika tepat waktu → status: PRESENT
```

## Clock Out

```
Karyawan pulang
  → Klik "Clock Out"
  → Sistem record waktu keluar
  → Hitung total jam kerja
```

## Koreksi Absensi (Supervisor/Owner)

```
Supervisor buka menu Absensi
  → Cari karyawan + tanggal
  → Edit clock in / clock out
  → Input alasan koreksi
  → Simpan → tercatat siapa yang koreksi
```

## Laporan Absensi (Owner)

```
Owner buka Laporan Kehadiran
  → Filter per karyawan / bulan
  → Lihat: hadir, terlambat, absen, cuti
  → Export PDF/Excel
```
