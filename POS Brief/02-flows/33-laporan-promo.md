# Flow: Laporan Promo & Diskon

**Role**: Owner

## Statistik Penggunaan Promo

```
Owner buka Laporan → Promo
  → Lihat per promo:
      → Berapa kali dipakai
      → Total diskon diberikan
      → Revenue dari transaksi yang pakai promo
  → Promo paling sering dipakai
  → Promo paling besar nilai diskonnya
```

## Efektivitas Promo

```
Owner buka Laporan → Promo → Efektivitas
  → Bandingkan revenue periode promo aktif vs tidak aktif
  → ROI promo: revenue tambahan vs diskon diberikan
  → Rekomendasi: promo worth it atau tidak
```
