# Gap Analysis — Fitur Qasir vs POS Mentai

## Fitur Qasir yang Sudah Ada di Dokumen Awal

| Fitur | Status |
|-------|--------|
| Kasir digital + print struk | ✅ Ada |
| Multi outlet/cabang | ✅ Ada |
| Manajemen stok & inventori | ✅ Ada |
| QRIS payment | ✅ Ada |
| Dashboard laporan lengkap | ✅ Ada |
| Role & hak akses pegawai | ✅ Ada |
| Absensi pegawai | ✅ Ada |
| Integrasi GrabFood | ✅ Ada |
| Export laporan | ✅ Ada |
| Analitik penjualan | ✅ Ada |
| Customer database | ✅ Ada |

## Fitur Qasir yang Ditambahkan

| Fitur | Keterangan |
|-------|------------|
| Membership & loyalty program | Poin, tier Bronze/Silver/Gold, redeem diskon & produk |
| Diskon otomatis/template promo | Multi trigger, auto & manual, buy X get Y |
| Kasbon & cicilan pembayaran | Hutang + jadwal cicilan, reminder jatuh tempo |
| Reminder expired product | Produk jadi & bahan baku, H-7/3/1, notif app+WA |
| Kitchen/order printer | Printer hanya di kasir, struk print/WA/skip |
| Status order workflow | Kustomisasi per outlet, customer bisa lihat & edit |
| Website/backoffice dashboard | Sudah tercakup (semua berbasis web app) |
| Multi harga/grosir | Per tier, per outlet, per waktu (happy hour) |
| Manajemen shift kasir | Buka/tutup shift, rekap kas, selisih otomatis |
| PO/down payment order | PO + DP, reserve stok, reminder pickup |
| Approval/transaksi tertentu | PIN + notif, per jenis transaksi, log audit |

## Tambahan di Luar Qasir

| Fitur | Keterangan |
|-------|------------|
| Bar role + approval flow | Kitchen → Bar approve → Kasir |
| Edit pesanan via QR meja | Customer edit + approval kasir/bar |
| Reservasi meja | Booking meja dengan jadwal |
| Split bill / gabung bill | Pemisahan & penggabungan tagihan |
| Resep / HPP otomatis | Bahan baku terhubung ke produk |
| Eskalasi approval | Auto eskalasi jika tidak direspons dalam X menit |
